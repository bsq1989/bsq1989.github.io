xgboost 安装

发表于 2019-06-18 | 分类于 机器学习

xgboost max os 下的配置

简介

根据官方网站自述，XGBoost是一套高效，灵活，便于移植的经过优化的分布式梯度提升库。它实现了梯度提升算法框架。它提供了一套行之有效的并行提升树算法。它部署在如hadoop，MPI之类的主流分布式系统上，可以解决具有数亿样本的大数据问题。
XGBoost问世以来，在kaggle等大数据比赛中取得了优异的成绩，是一款主流的机器学习框架。

XGBoost on MacOS

近期在自己的Mac笔记本上部署了XGBoost，将过程记录如下。
XGBoost的安装指导非常友好，达到了Step by Step 的程度，本文也是基本搬运网站内容，具体文档详见[1]。

  • XGBoost 是一款具备并行处理能力的算法库，因此，如果本机不具备可以编译openMP的g++,需要先安装gcc，在mac下，利用brew 安装g++，代码如下brew install gcc@8
  • 从git克隆xgboost代码库，注意，这里最好利用命令行下载代码，通过网页界面下载可能确实部分代码，切换到目标路径，输入git clone --recursive https://github.com/dmlc/xgboost
  • 下载完成后，切换至代码路径，首先建立build文件夹，mkdir build
  • 切换至build文件夹，输入CC=gcc-8 CXX=g++-8 cmake .. 生成make文件
  • 输入make -j4 等待编译完成即可
  • 编译完成后，会生成dlmc.dylib，后续准备利用python进行算法开发，因此进入python-package文件夹进行算法库部署。
  • 输入如下指令cd python-package;sudo python setup.py instal进行环境部署，会部署相关模块至python环境中。
    至此，环境部署完毕。
